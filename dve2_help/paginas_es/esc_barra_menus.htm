Menú Elemento

Convierte un texto plano en un texto con estilo.

En el texto plano no se pueden editar partes de él sino que la configuración (tipo de letra, tamaño, color, etc.) es uniforme para todo el texto.

El texto con estilo se puede editar y podemos encontrar distintos formatos dentro del mismo texto.

  y    Convierte un objeto genérico en específico y viceversa.

Los objetos específicos son los que sirven para mostrar un único tipo de contenido multimedia en el escenario, y no se usan como elementos principales de las interacciones.

Los objetos genéricos pueden incluir varios contenidos multimedia (o ninguno si queremos crear una zona activa), y son los que se usan para definir las partes interactivas de una actividad (los huecos en un ejercicio de completar, las opciones que hay asociar con otras, etc.)
