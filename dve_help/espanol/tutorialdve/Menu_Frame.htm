Menú del Editor Malted (DVE) - Modo Editor de Frames

Menú "Archivo":

Nuevo frame: Abre el cuadro de diálogo "Nuevo frame". El usuario puede seleccionar una plantilla (modelo de frame) del menú desplegable y tambien previsualizarla antes de añadirla al proyecto. Por defecto está seleccionada la plantilla "_Blank" que no tiene asociada ninguna interacción. El Nombre del Frame se puede cambiar o dejar el que viene por defecto. Se puede tener a la vez más de un frame abierto.

Abrir...: SIrve para abrir un frame ya existente en el Proyecto. Abre un cuadro de diálogo con una lista de los frames que contiene el Proyecto, de entre los cuales se puede escoger uno. Puede haber más de un Frame abierto a la vez.

Cerrar: Cierra el frame activo. Si no ha sido guardado, el sistema pregunta si se quiere guardar.

Guardar: Guarda el Frame en el disco duro con el Nombre que se le había asignado previamente al crearlo.

Guardar como...: Permite guardar el Frame con un nuevo Nombre, manteniendo una copia del Frame con el Nombre original.

Importar frame: Abre un cuadro de diálogo para importar Frames de otros Proyectos al nuestro en el que estamos trabajando.



La Importación Remota aún no está implementada. Hay que pulsar el botón "Buscar en discos locales", localizar el Proyecto con el frame deseado y dentro de la carpeta xml del Proyecto seleccionar el Frame y abrirlo. Se puede repetir la operación para importar más Frames a la lista y cuando se haya terminado se debe pulsar el botón "Agregar la lista al proyecto" para que se realice la importación.

Nuevo proyecto: Abre el cuadro de diálogo de "Nuevo proyecto" en el que el usuario debe introducir el Nombre del nuevo proyecto. Ésta opción está desactivada si ya hay algun Proyecto abierto en el Editor Malted. Para poder crear un Proyecto nuevo es necesario cerrar de antemano el Proyecto en el que se esté trabajando. El cuadro de diálogo tambien permite elegir la ubicación en el disco duro donde se desea guardar.

Abrir proyecto: Abre un cuadro de diálogo en el que hay una lista con los proyectos existentes en el disco duro de entre los cuales el usuario puede escoger uno. Para abrir un proyecto, seleccionarlo y luego pulsar el botón Abrir.

Cerrar proyecto: Cierra el proyecto que está abierto. Pregunta si se desean guardar los cursos o los frames del proyecto que no han sido guardados.

Salir: Cierra la aplicación. Pregunta si se desea guardar los cursos o los frames que no han sido guardados. Guarda las opciones de la aplicación para usos posteriores.

Menú "Edición"

Deshacer: Deshace el último cambio realizado en el Frame en el que se está trabajando.

Cortar / Pegar / Copiar / Borrar: Acciones básicas de edición aplicables a todos los Objetos del Frame (Objeto Media, Paneles, Tarjetas, Contenedores).

Editar Script : Abre el código xml del Frame en el editor de texto para poder modificarlo. Para que ésta opción se active es necesario Guardar primero los últimos cambios realizados en el frame. Una vez hechos los cambios en el código xml, guardar el fichero xml y al volver a la ventana del Editor Malted se debe seleccionar el menú Edición > Actualizar frame para que se reflejen los cambios en la ventana de edición del Frame.

Actualizar Frame: Actualiza el frame cuando hemos realizado cambios en el código xml para que se reflejen los cambios en la pantalla de edición.

Preferencias: Abre la ventana "Malted Opciones" que permite cambiar el idioma de la aplicación, establecer si deseamos que al iniciar el Editor Malted se abra en Modo Editor de Frames o de Curso, añadir aplicaciones externas al menú de Herramientas, y configurar el acceso a una Base de Datos.



Menú "Frame"

 Las acciones de este menú están disponibles sólo en el caso de que haya algún frame activado.

Propiedades: Abre la ventana de Propiedades del frame, donde se asignan la Interacción y otras características generales del Frame. Para más información pulsar aquí.

Descripción: Abre el panel de propiedades de los frames en el que se encuentra el Nombre del frame y una descripción del mismo que se puede modificar.

Ejecutar frame: Abre el Frame en modo de Previsualización con todas las Acciones funcionales salvo aquellas que hagan referencia a otros frames del Curso.

Menú "Media"

 Importar Media: permite localizar archivos Media que están fuera de nuestro Proyecto para incorporarlos. Abre un cuadro de inserción desde el que se puede seleccionar un fichero media del disco duro o cargar un fichero media de una base de datos externa.



Pulsamos en Buscar en discos locales y localizamos los ficheros a importar.



Al hacer doble click sobre un fichero se añade a la lista de importación y tenemos la posibilidad de repetir el proceso con más ficheros.



Cuando hayamos terminado de agregar ficheros a la lista pulsamos Agergar la lista al proyecto. Los archivos de la lista se copiaran en las carpetas correspondientes (imagen, audio, etc) del proyecto.

Edición: Inicia un editor externo para editar el archivo media seleccionado. Se escoge la aplicación según el tipo de formato que tenga el archivo media, tal como está definido en el apartado "Opciones".

Previsualización: Proporciona una vista preliminar del archivo media seleccionado.

Renombrar: Abre un cuadro de diálogo con un campo de texto lleno que corresponde al nombre del archivo media actual. Se puede escribir un nombre nuevo en el campo de texto y confirmar la acción haciendo clic sobre la opción "Aceptar".

Menú "Herramientas"

Ofrece una lista de los editores disponibles (aplicaciones externas) que han sido definidos previamente en el menú Edición > Preferencias. Cada apartado de este menú abre la aplicación correspondiente.

Menú "Modo"

Curso: Activa el Modo Editor de Cursos.

Frame: Activa el Modo Editor de Frames.

En la segunda mitad del menú Modo aprece una lista con los Frames abiertos. El frame activo aparece con un punto. Desde aquí se puede cambiar de frame activo para editar otro frame que esté abierto.

Menú "Ayuda"

Ayuda Malted: Abre una ventana con el tutorial del Editor Malted (DVE).

Acerca de...: Abre una ventana con información sobre el proyecto Malted.
