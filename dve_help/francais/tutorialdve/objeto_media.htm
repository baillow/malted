4. L´objet média:

Objet média: il est représenté par un carré gris au moment de sa création. C ´est l´outil qu´on utilisera pour introduire du texte, des images, audio, vidéo, et des actions comme par exemple des liens ou la reproduction multimédia. C ´est donc l´élément basique et fondamental pour créer un Frame.

1. Création: pour ajouter un Objet Média, tout en étant dans le mode édition de Frames :
- Bouton droit de la souris sur la zone de travail> insérer le média

2. Edition: double cliquer sur l´objet média ou on le choisit avec le bouton droit et on appuie sur l´option Edition du menu. La fenêtre d´Edition de l´Objet Média s´ouvrira. Sur la case supérieure on pourra assigner un nom. En dessous il y a trois Onglets qui permettent d´éditer l´objet:

2.1 Onglet Contenu: à partir d´ici nous pouvons assigner le contenu, l´action et autres Propriétés:

Autrement le fichier n´apparaîtra pas sur le dépliable sur la droite de chaque type de fichier. Pour introduire ces archives dans leurs dossiers respectifs l´utilisateur peut le faire à partir de l´explorateur du système, à la main, ou dans l´éditeur Malted, à partir du mode d´édition de Frames : menu Media> Importer média qui permet de repérer les fichiers souhaités dans le disque dur ou dans le réseau et de les importer dans leurs dossiers respectifs dans notre Projet.

Contenu:
1. Texte: permet d´introduire un paragraphe de texte avec des sauts de ligne ; il partira du bord supérieur de la caisse.(caja)
2. Etiquette: le texte introduit ici apparaîtra centré verticalement et sur une seule ligne.
3. Fichiers: pour associer à un Objet média un fichier déjà créé (texte, image, audio, vidéo)celui-ci doit d´abord être dans le dossier adéquat dans l´Unité en question (audio, image, texte, vidéo). Autrement le fichier n´apparaîtra pas sur le dépliable sur la droite de chaque type de fichier. Propriétés du Média :
- transparent : change le fonds du média à transparent et élimine le cadre fin qui par défaut entoure l´Objet Média.
- Case de vérification : ajoute une case de vérification sur le côté droit du média qui s´active ou se désactive avec le bouton gauche de la souris.
- Zone éditable : transforme le contenu de la fenêtre Texte en zone éditable par l´utilisateur.
- Incruster : n´a de sens que dans le cas où nous avons associé un fichier audio ou vidéo à l´objet média. Avec cette option sélectionnée un tableau de contrôle de l´audio/vidéo apparaîtra aux yeux de l´utilisateur.

Action et valeur du Média :
Il existe de multiples actions que l´on peut assigner à un Objet Média pour qu´en cliquant avec la souris sur lui, l´action sélectionnée s´exécute. Créer des liens dans le Projet, ouvrir des applications externes, exécuter des audios et des vidéos… Pour voir la liste complète avec des exemples cliquez ici. Pour les Actions qui requièrent une Valeur déterminée, il apparaît en-dessous de la fenêtre Valeur les instructions pour remplir cette case.

 2.2 Onglet Etiquettes: permet de contrôler l´aspect de la police, son format et la couleur du fonds dans l´Objet Média. Ces propriétés peuvent être sujettes à celles définies dans le Container qui contient l´Objet Média.

 

 

2.3 Onglet évènements: Permet d´assigner une action Malted à un Evènement de la souris, par exemple que la petite flèche représentant la souris sur l´écran (puntero) passe par dessus de l´Objet Média (onMouseOver) ou que l´on clique sur l´Objet Média avec le bouton droit (onRightClick). A droite de chaque évènement il y une case où l´on devra introduire une valeur pour les Actions qui le requièrent. Si l´on n´est pas sûr de la syntaxe, on peut la vérifier pour cette Valeur en choisissant la même Action dans le Onglet Contenu.
