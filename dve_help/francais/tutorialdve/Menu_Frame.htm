Menu de l’Editeur Malted (DVE) – Mode Editeur de Frames

Menu "Fichier":

Nouveau frame: Ouvre la fenêtre de dialogue "Nouveau frame". L’utilisateur peut sélectionner un modèle (modèle de frame) du menu qui se déplie et peut aussi le prévisualiser avant de l’ajouter au projet .
Le modèle " _Blank" qui n’a aucune interaction liée, est sélectionné par défaut. Le Nom du Frame peut être changé ou bien on peut conserver celui qui apparaît par défaut.
On peut avoir plusieurs frames ouverts en même temps.

Ouvrir...: Sert à ouvrir un frame qui existe déjà dans le Projet. Il ouvre un fenêtre de dialogue avec une liste des frames qui contiennent le Projet, parmi lequels on peut choisir un.
Il peut y avoir plusieurs Frames ouverts en même temps.

Fermer: Ferme le frame actif. S’il n’a pas été sauvé, le système demande si on veut le sauver.

Sauver: Sauve le frame dans le disque dur avec le Nom qu’on lui avait assigné au début lorsqu’on la créé.

Sauver sous... : Permet de sauver le Frame avec un Nom nouveau, en gardant une copie du Frame avec le Nom original.

Importer frame: Ouvre une fenêtre de dialogue pour importer des Frames d’autres projets vers celui sur lequel on est en train de travailler



L’Importation Remota n’est pas encore développée. Il faut appuyer sur le bouton "Chercher dans les disques locaux", localiser le projet avec le frame souhaité et sélectionner le Frame dans le dossier xml du dossier pour l’ouvrir ensuite. On peut répéter la même démarche pour importer plus de Frames à la liste. Lorsqu’on aura finit, il faut appuyer sur le bouton "Ajouter la liste au projet" pour que l’importation se fasse.

Nouveau projet: Ouvre la fenêtre de dialogue "Nouveau Projet" dans laquelle l’utilisateur doit introduire le Nom du nouveau projet. Cette option est désactivée s’il y a déjà un projet ouvert dans l’Editeur Malted. Pour pouvoir créer un Projet, il faut tout d’abord fermer le Projet sur lequel on est en train de travailler. La fenêtre de dialogue permet aussi de choisir l’emplacement dans le disque dur.

Ouvrir projet: Ouvre une fenêtre de dialogue dans laquelle il y a une liste avec les projets existants dans le disque dur, parmi lesquels l’utilisateur peut en choisir un. Pour ouvrir un projet, il faut le sélectionner et ensuite appuyer sur le bouton Ouvrir.

Fermer Projet: Ferme le projet qui est ouvert. Il demande si on veut sauver les cours ou les frames du projet qui n’ont pas été sauvés.

Sortir : Ferme l’application. Il demande si on veut sauver les cours ou les frames qui n’ont pas été sauvés.

Menu "Edition"

Défaire: Défait le dernier changement réalisé dans le Frame dans lequel on est en train de travailler.

Couper / Coller / Copier / Effacer: Ces actions basiques de l’édition applicables à tous les Objets du Frame (Objet Média, Panneaux, Cartes, Conteneurs).

Editeur Script: Ouvre le code xml du Frame dans l’éditeur de texte pour pouvoir le modifier. Pour activer cette option, il faut tout d’abord Sauver les derniers changements réalisés dans le cours.
Après avoir fait les changements dans le code, sauver le fichier dans l’éditeur de texte, et de retour à la fenêtre de l’Editeur Malted, on doit sélectionner le menu Edition > Actualiser frame pour que les changements soient saisis dans le Frame.

Actualiser Frame: Actualise le frame lorsqu’on a réalisé les changements dans le code.

Préférences: Ouvre la fenêtre "Malted Options" qui permet de changer la langue de l’application. Permet aussi d’établir si on veut, que lorsqu’on démarre l’Editeur Malted, celui-ci s’ouvre en Mode Editeur de Frames ou de Cours. Il permet de même d’ajouter des applications externes au menu Outils, et configurer l’accès à une Base de Données.



Menu "Frame"

 Les actions de ce menu sont seulement disponibles au cas où il y a un frame activé.

Propriétés: Ouvre la fenêtre de propriétés du frame, où l’on assigne l’Interaction et d’autres caractéristiques générales du frame. Pour plus de renseignements appuyer ici.

Description: Ouvre le panneau de propriétés des frames dans lequel se trouve le Nom du frame et une description de celui-ci qui peut être modifiée.

Exécuter frame: Ouvre le Frame en mode Prévisualisation avec toutes les Actions fonctionnelles sauf celles qui font allusion à d’autres frames du Cours.

Menu "Media"

 Importer Média: Permet de localiser les fichiers Média qui sont hors notre Projet pour les incorporer.
Il ouvre une fenêtre d’insertion deppuis laquelle on peut sélectionner un fichier média du disque dur ou charger un fichier média dans une base de données externe.



On appuie sur Chercher dans les disques locaux et on localise les fichiers à importer.



En faisant double clic sur un fichier, il s’ajoute à la liste d’importation. On peut répéter le même processus avec plus de fichiers.



Lorsqu’on aura finit d’ajouter des fichiers à la liste, on appuie sur Ajouter à la liste du projet. Les fichiers de la liste seront copiés dans les dossiers correspondants (image, audio, etc…) du projet.

Edition: Démarre un éditeur externe pour éditer le fichier média sélectionné. On choisit l’application en fonction du type de format qu’a le fichier média, comme définit dans le paragraphe "Options".


Prévisualisation: Fournit une vue préliminaire du fichier sélectionné.

Renommer: Ouvre une fenêtre de dialogue avec un espace de texte remplit avec le nom qui correspond à celui du fichier du média actuel. On peut écrire un nom nouveau dans l’espace de texte et le confirmer en faisant double clic sur l’option "Accepter".

Menu "Outils"

Ouvre un liste aves les éditeurs disponibles (application externes) qui ont été définis préalablement dans le menu Edition > Préférences. Chaque paragraphe de ce menu ouvre l’application qui lui correspond.


Menu "Mode"

Cours: Active le Mode Editeur de Cours.

Frame: Active le mode Editeur de Frames.

Dans la deuxième moitié du menu Mode apparaît une liste avec les Frames ouverts. Le frame actif apparaît avec un point. C’est à partir de là, que l’on peut changer le frame actif pour éditer un autre frame qui soit ouvert.

Menu "Aide"

Aide Malted: Ouvre une fenêtre avec le tutoriel de l’Editeur Malted (DVE).

A propos de… : Ouvre une fenêtre avec des renseignements sur le projet Malted.
