5. Le Conteneur :

 Le Conteneur: représenté par un cadre bleu : il délimite une couche avec des dimensions et des coordonnées qui peuvent être configurées. Il a les fonctions suivantes:

A) Il s’emploie pour conférer des coordonnées de position aussi bien à des Objets Média comme à des Panneaux, et avoir ainsi plus de contrôle sur son positionnement . Il suffit d’introduire le Média ou le Panneau à l’intérieur d’un conteneur et ensuite éditer le conteneur pour modifier les coordonnées de positionnement sur l’écran.

B) Il permet d’uniformiser les dimensions de tous les médias à l’intérieur du conteneur.

C) On peut lui assigner un nom, ce qui est nécessaire pour employer les Intéractions programmées dans Malted.


1. Création: Pour ajouter un Conteneur, en étant sur le mode édition de Frame:
- Bouton droit de la souris sur la région de travail > Insérer un conteneur

 2. Edition: on fait double clic sur le Conteneur, on le sélectionne avec le bouton droit (lorsqu’on sélectionne le conteneur, son cadre passe du bleu au jaune) et on appuie sur l’option Edition du menu. La fenêtre d’édition du conteneur s’ouvrira . Dans la caise supérieure, on peut assigner un nom. Dessous, il y a deux cils qui permettent de configurer les propriétés du conteneur.



Onglet Dimensions :

- Limites du Conteneur: Les coordonnées de Gauche et du Haut sont des coordonnées du coin supérieur gauche du Conteneur. Hauteur et Largeur définissent les dimensions du conteneur. Ces numéros peuvent s’introduire ici ou peuvent aussi s’ajuster graphiquement en sélectionnant le conteneur, en plaçant la flèche de la souris sur les bords et en tirant les flèches de position et dimension .


- Taille des objets intérieurs: Pour que cette option soit éditable, la case « Uniform » doit être marquée.
Il permet de préciser la Hauteur et la Largeur qui s’appliqueront à tous les Objets Média inclus à l’intérieur du Conteneur leur donnant une taillant uniforme.

 
Si la hauteur et la Largeur ne sont pas spécifiés, l’uniformisation se faira aussi bien du côté de la Largeur comme de la Hauteur jusqu’à remplir l’intérieur du Conteneur donnant lieu à des caisse de 2x2, 3x3, etc :
Le bouton Remplir ajuste les dimensions des Objets Médias, en arrivant jusqu’à la limite intérieure du Conteneur .
- Autres propriétés :
Transparent: Il permet le fond du conteneur d’être transparent. Cette propriété est héritée par tous les Objets Média qui sont contenus, mais c’est possible d’éditer postérieurement chaque Média indépendamment por contrôler sa transparence.

Déploiement aléatoire: Il permet les Objets Média qui sont à l’intérieur du Conteneur se montrent dans un ordre aléatoire. Cette propriété aléatoire ne se montre pas dans la zone de travail de l’Editeur Malted, mais en exécutant le frame en question. (Pour pouvoir l’exécuter il faut le Sauver auparavant).

Déploiement Checkbox: Il rajoute à gauche de chaque Objet Média à l’intérieur du Conteneur une case de vérification. Tous les Médias qui y sont contenus héritent cette propriété. On peut après éditer chaque Média individuellement pour désactiver cette option si on le souhaite.

Onglet Balises:
Il permet de contrôler l’aspect de la Source, son format et la couleur de fond à l’intérieur de tous les Objets Média inclus dans le Conteneur. Ces propriétés peuvent se modifier plus tard, de façon indépendante pour chaque Objet Média, si on ne souhaite pas une uniformité totale des styles dans tous les Médias.


Superposition des Conteneurs:
Lorsque nous avons plusieurs Conteneurs ou des zones de ceux-ci superposées, si on veut modifier l’ordre des couches il faut éditer le code xml du frame. Pour ceci, il faut d’abord Sauver le Frame et ensuite appuyer sur l’icöne XML ou sélectionner le menu Edition > Editer script. Le code xml s’ouvre par défaut avec le WordPad en Windows. Les Conteneurs apparaissent dans le code compris entre balises
<BOUNDS></BOUNDS>, par exemple :

<BOUNDS NAME="new" UNIFORM="true" LEFT="413" TOP="365" WIDTH="100" HEIGHT="100" RANDOM="false">
</BOUNDS>

Les Conteneurs s’ordonnent en couches en fonction de leurs positionnement dans le code. Ceux qui apparaissent avant dans le code, se trouvent dans des couches supérieures que ceux qui apparaissent plus bas dans le code. Par conséquent, changer la couche dans laquelle se trouve le Conteneur est aussi simple que changer sa situation dans le code par un copier- coller.

Une fois le code modifié, Sauver le document xml dns le WordPad et revenir à la fenêtre de Malted. Sélectionner le menu Edition > Actualiser frame et les changements réaliser dans le code vont s’appliquer.
