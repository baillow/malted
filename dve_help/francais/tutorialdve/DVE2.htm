2. L’Editeur Malted (DVE). Mon premier Projet

L’Editeur Malted (DVE) peut travailler sous 2 modes différents:

A) Mode Edition de Frames: C’est le mode qui s’emploie pour créer et éditer des Frames. Il a son propre menu, et il montre à gauche les dossiers d’image, audio, vidéo et texte du Frame sur lequel on est en train de travailler. En faisant double clic sur ces dossiers le contenu de chaquun d’eux se déploie, ce qui permet de prévisualiser les fichiers et de changer leurs noms.

B) Mode Edition des Cours : On l’emploie pour créer un Cours, après avoir créer tous les Frames qui vont constituer le Cours. Il a aussi son propre menu. Il montre à gauche la liste de tous les Frames inclus dans le projet dans lequel on est en train de travailler. En faisant double clic sur un Frame de la liste, celui-ci s’éxécute en mode prévisualisation.
Si on chosit un frame de la liste et on appuie ensuite sur le bouton droit de la souris, l’option Editer du frame apparaît ( le passage au Mode Edition de Frames se fait automatiquement).

Pour passer d’un Mode à un autre, il faut appuyer sur l’icone ou sélectionner le menu Mode > Cours ou bien Mode > Frame.

Dans la barre de navigation supérieure de la fenêtre apparaît le Mode dans lequel on se trouve à chaque moment.

1. Lorsqu’on ouvre l’Editeur Malted (DVE), pour pouvoir commencer à trvailler, c’est nécessaire de créer un Projet Nouveau ou d’ouvrir un projet qui existe déjà.. Ceci se fait à partir du menu. Ceci se fait dans le menu Archive > Nouveau Projet ou Archive > Ouvrir projet



2. On choisit dans le menu Archive > Nouveau projet. Une nouvelle fenêtre apparaît. On introduit là le Nom du Projet. Dans cet exemple, on écrit "Preuve". Lorsqu’on assigne un nom, il faut tenir compte qu’il ne peut y avoir des espaces blancs ni des caractères bizarres. On appuie sur Accepter.




3. l’Editeur malted entre dans le Mode Editeur de Frames par défaut. Le pas suivant consiste à créer un Frame en apputyant sur l’icone "Nouveau Frame" ou dans le menu Archive > Nouveau frame.



4. Une fenêtre apparaît dans laquelle on peut choisir le modèle sur laquelle on veut que le frame soit basé. Une petite description et le Nom du Frame éditable apparaissent aussi dans une case. Le modèle par défaut est "_Blank.template" (modèle vide).



5. Pour ce premier exemple, on choisit le modèle "MCh_1Q_anyTime.template". Il existe la possibilité de Prévisualiser le modèle pour savoir s’il nous intéresse, et de changer le Nom du Frame qui vient par défaut. On appuie sur Accepter pour continuer.



6. Pour savoir comment ces éléments sont édités, il faut voir le paragraphe suivant : " 3. Composantes basiques pour créer un frame "
Pour prévisualiser le frame et son fonctionnement, il faut appuyer sur l’icone "Exécuter frame" ou sélectionner le menu Frame > Exécuter frame. Dans ce mode de Prévisualisation du frame, toutes les actions sont fonctionnelles sauf celles qui font le lien avec d’autres frames du Cours.
Si on veut essayer la navigation entre Frames du Cours, on doit le faire à partir du Mode Cours comme on verra plus tard dans le point 12.

7. On va créer maintenant un nouveau frame à partir d’un autre modèle pour l’ajouter au projet. On choisit Archive> Nouveau frame ou bien on appuie sur l’icone Nouveau frame et dans le fenêtre qui ressort , on choisit le modèle "dialog_000.template", et on appuie sur Accepter. 9. On appuie sur l’icone Nouveau cours ou bien on choisit Archive> Nouveau cours.
On introduit le même nom pour le Cours que pour le Projet : "preuve". Ce n’est pas nécessaire que Projet et Cours aient le même nom, mais c’est quelque chose qu’on fait d’habitude pour simplifier l’usage du Projet.
On appuie sur Accepter.



10. On observe que sur la barre supérieure, on nous indique qu’on se trouve dans le projet preuve. Mode Edition de COURS, et cours preuve.cxml.



11. 10. On choisit avec le bouton gauche de la souris le Frame "MCh_1que_anyTime.xml" et on tire de lui jusqu’à le placer dans la fenêtre de droite. C’est là que s’établit l’ordre que vont suivre les Frames (écrans) du Cours. On refait postérieurement la même opération avec le Frame " Dialog_000.xml ". L’ordre des Frames dans le Cours suit une numération ascendante, commençant par 0,1,2…
Si on sélectionne maintenant un des Frames de la fenêtre de droite, on peut le faire monter ou descendre en suivant la séquence linéaire des Frames en employant les boutons "Haut" et "Bas".




12. Pour vérifier le fonctionnement correct du Cours, on doit d’abord appuyer sur l’icône Sauver, ou sélectionner le menu Archive > Sauver. Lorsqu’on aura sauver les changements, on sélectionne le menu Cours > Exécuter le cours. Ceci lance Malted RTE*, un Navigateur qui incorpore des options d’édition pour les développeur d’Unités didactiques.



13. Lorsqu’on éxecute le Cours, l’application “Navigateur Editeur Malted (RTE) *” s’ouvre et on peut vérifier le fonctionnement de tous les Objets Média, Cartes, etc, qui sont incorporés.
Le bouton Verify, vérifie les réponses sélectionnées dans la carte actuelle. Si on appuit sur la flèche qui amène au Frame suivant, l’écran avec l’exercice Dialog_000 devrait s’ouvrir.
