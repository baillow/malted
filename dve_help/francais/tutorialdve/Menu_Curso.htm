Menu de l’Editeur Malted (DVE) – Mode Editeur de Cours

Mode Editeur de Cours:

Menú "Fichier":

Nouveau cours : Crée un cours nouveau ; S’il ya un cours actif dans la fenêtre cours, il le ferme en demandant d’abord si on souhaite le sauver.

Ouvrir… : Ouvre une fenêtre de dialogue ayant une liste des cours qui contiennent le Projet, parmi lesquels on peut choisir un. Au contraire que pour les frames, il ne peut y avoir plus d’un cours activé en même temps.

Fermer : Ferme le cours activé. S’il n’a pas été sauvé, le système demande si on veut le sauver avec l’extension qui lui correspond ( .cxml) dans le Projet.

Sauver : Sauve le cours avec l’extension .cxml dans le Projet.

Sauver sous… : Sauve le cours avec l’extension .cxml dans le disque dur et avec un nom de fichier différent.

Importer cours : Ouvre une fenêtre de dialogue pour importer des cours d’autres projets.

Nouveau projet: Ouvre la fenêtre de dialogue de "Nouveau Projet" dans lequel l’utilisateur doit introduire le Nom du nouveau projet. Cette option n’est pas activée s’il y a déjà un Projet ouvert dans l’Editeur Malted. Pour créer un Projet nouveau, il faut préalablement fermer le Projet dans lequel on est en train de travailler.
La fenêtre de dialogue permet aussi de choisir l’emplacement du disque dur dans lequel on souhaite le placer.

Ouvrir projet: Ouvre une fenêtre de dialogue dans laquelle il y a une liste avec tous les projets existants dans le disque dur parmi lesquels l’utilisateur peut en choisir un. Pour ouvrir un projet, il faut d’abord le sélectionner et ensuite appuyer sur le bouton Ouvrir.

Fermer projet: Ferme le projet qui est ouvert. Demande si on veut sauver les cours ou les frames du projet qui n’ont pas été sauvés.

Renommer: Ouvre une fenêtre de dialogue dans laquelle on peut écrire un nom nouveau pour le cours sélectionné. Un click sur l’option "Accepter" ouvre une fenêtre de dialogue qui demande de confirmer l’action à exécuter. Au cas ou l’on ne remplirait pas le nom du projet, un message d’erreur apparaîtrait sur l’écran.

Sortir: Ferme l’application. Demande si on veut sauver les cours ou les frames qui n’ont pas été sauvés.
Sauve les options de l’application pour des utilisations postérieures.

Menu "Edition"

Défaire: Défait le dernier changement réalisé dans le cours dans lequel on est en train de travailler.

Effacer: Efface le Frame sélectionné dans la fenêtre de gauche.

Editer Sript: Ouvre le code cxml du Cours dans l’Editeur du texte pour pouvoir le modifier. Pour activer cette option, il faut tout d’abord Sauver les derniers changements réalisés dans le cours.
Après avoir fait les changements dans le code, sauver le fichier dans l’éditeur de texte, et de retour à la fenêtre de l’Editeur Malted, on doit sélectionner le menu Edition > Actualiser frame pour que les changements soient saisis dans le Cours.

Actualiser Frame: Actualise le Cours lorsqu’on a réalisé les changements dans le code.

Préférences: Ouvre la fenêtre "Malted Options" qui permet de changer la langue de l’application. Permet aussi d’établir si on veut, que lorsqu’on démarre l’Editeur Malted, celui-ci s’ouvre en Mode Editeur de Frames ou de Cours. Il permet de même d’ajouter des applications externes au menu Outils, et configurer l’accès à une Base de Données.

Menu "Cours"

Description: Ouvre une fenêtre avec les propriétés du cours, d’où on peut modifier le nom, la description et la structure (linéaire ou complexe).

Ajouter un frame: Ouvre une fenêtre de dialogue avec la liste des modèles. Le frame sélectionné sera ajouté au frame actif.

Exécuter le cours: Ouvre le Navigateur Editeur Malted (RTE) en permettant de surfer dans le cours er dans tous les frames .

Create Course html: Crée un fichier html du cours dont on a va s’en servir pour pouvoir le voir à travers l’applet Malted de la web.

Menu "Frame"

Importer frame: Ouvre une fenêtre de dialogue pour importer des Frames d’autres projets différents à celui sur lequel on est en train de travailler.



L’Importation Remota n’est pas encore développée. Il faut appuyer sur le bouton "Chercher dans les disques locaux", localiser le projet avec le frame souhaité et sélectionner le Frame dans le dossier xml du dossier pour l’ouvrir ensuite. On peut répéter la même démarche pour importer plus de Frames à la liste. Lorsqu’on aura finit, il faut appuyer sur le bouton "Ajouter la liste au projet" pour que l’importation se fasse.

Description: Ouvre le panneau de propriétés des frames dans lequel se trouve le Nom du frame et une description de celui-ci qui peut être modifiée.

Editer frame: Ouvre le frame sélectionné dans la fenêtre gauche, en passant de façon automatique au Mode Editeur des Frames.

Editer link:

Exécuter frame: Ouvre le Frame en mode Prévisualisation avec toutes les Actions fonctionnelles sauf celles qui font allusion à d’autres frames du Cours.

Menu "Mode"

Cours: Active le Mode Editeur de Cours.

Frame: Active le Mode Editeur de Frames.

Dans la deuxième moitié du menu Mode apparaît une liste avec les Frames ouverts. Le frame actif appqraît avec un point. C’est à partir de là, que l’on peut changer le frame actif pour éditer un autre frame qui soit ouvert.

Menu "Aide"

Aide Malted: Ouvre une fenêtre avec le tutoriel de l’Editeur Malted (DVE).

A propos de… : Ouvre une fenêtre avec information sur le projet Malted.
