7. Propriétés d’un Frame :

Un Frame est un écran du projet dans lequel on introduit Objets Média (avec images, textes, sons, et vidéos) et dans lequel on peut assigner une Intéraction, qui donnera de la fonctionalité à la page, comme Multiple Choice, Gap Filling, etc.

Dans le bouton Propriétés du Mode Frame de l’Editeur Malted (DVE), ou dans le menu Frame > Propriétés, on peut accéder à ces intéractions et à d’autres paramètres intéressants (taille de l’écran, image de fond, etc…) que l’on précise ci-dessous:


Le nom du frame peut être édité dans la case supérieure de la fenêtre Propriétés.
1. Onglet Description:
Contient un menu qui peut se déployer dans lequel on choisit l’Interaction souhaitée pour le frame. Dessous il y a une description de l’Interaction, des Modèles qui l’emploient et des éléments qu’il faut inclure dans le frame pour que l’Interaction soit fonctionnelle.


2. Onglet Paramètres:
Permet de réaliser les réglages suivants:
A) Définir la taille du frame et si l’on veut qu’il apparaisse en écran complet..
B) Choisir les sons de Succès et Erreur parmi tous les sons inclus dans le dossier audio du projet en question.
C) Ajuster les Valeurs des Paramètres de l’Interaction. Chaque Interaction a des paramètres spécifiques dont on peut modifier les valeurs. Par exemple, l’Interaction Memory, qui reproduit le jeu classique de mémoire dont l’objectif consiste à mettre deux par deux les cartes qui sont tournées, a pour paramètre l’image que l’on veut qu’ait l’envers des cartes (backCardImage).

3. Onglet Image de fond :
Permet d’assigner une image de fond à tout le Frame et sélectionner la forme que l’on souhaite soit déployée (centré, répété, ou étiré). Esto puede resultar muy útil si queremos diseñar el Frame en un programa de dibujo y guardarlo como una imagen, para luego importarla a Malted. Tras insertar la imagen como imagen de fondo en un frame de Malted, podemos añadir objetos media transparentes encima de distintas área del dibujo, aportando funcionalidad y Acciones.

4. Pestaña Etiquetas:
El aspecto de las Etiquetas determinado aquí lo heredan los Objetos Media creados posteriormente. Se pueden asignar las siguientes características:
- el aspecto de la fuente (fuente, tamaño, color, estilo, alineación)
- el color de fondo dentro del Objeto Media. Estas propiedades pueden estár supeditadas a las definidas en el Contenedor que contiene al Objeto Media.
- el color de hypervínculos en su estado de visitado y no visitado. Para información acerca de cómo crear estos hypervínculos pulsar aquí y seguidamente pulsar "hypertext".

5. Pestaña Ac.Auto.:
Permite lanzar una acción de forma automática al abrirse el Frame en cuestión (un popout, un audio, vídeo, etc.). Será necesario rellenar la casilla de la derecha con un Valor adecuado. (Ver Acciones)
