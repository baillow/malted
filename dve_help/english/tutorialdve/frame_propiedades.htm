7. Frame Properties:

A Frame is a screen of the Project to which we add Media Objects such as image, text, audio and video files and to which we apply an Interaction that provides the page with functionality, such as Multiple Choice, Gap Filling, etc .

Click on the Behaviour icon in the Frame Management or select Frame>Behaviour in order to personalize both these interactions and any other interesting parameters, such as screen size, background image, which are described below :

The name of the frame can be edited in the top box from the Behaviour Window.
1. Description Thumb:
It includes an unfolding menu from which the designer can choose the Interaction which will be applied to the frame. Below, there is a description of the interaction, together with the templates used and the items which must be added to the frame so that the interaction is functional .


2. Parameters Thumb:
It allows the designer to personalize the following features :
A) Set up the size of the frame and if we would like it to be full screen.
B) Choose the sounds which will be associated to the Correct and Wrong audio feedback out of all the sounds included in the audio folder of the project .
C) Adjust the Values of the Interaction parameters. Each interaction has specific parameters whose values we can modify. For instance, the parameter associated to the image of the back of the card (backCardImage) in the Memory Interaction, which reproduces the classical memory game whose goal is to match cards placed backside .

3. Background Image Thumb:
It applies a background image to the whole frame and selects the way in which we want it to be laid out, such as centred, patched or stretched. This can be useful if we would like to design the frame with a drawing program and save it as an image that should be later imported to Malted. After inserting the image as a Background Image in a Malted frame, we can add transparent media objects to the various drawing areas, providing in that way both functionality and Actions.

4. Label Properties Thumb:
The labels outlook that we predetermine in this thumb will be applicable to all the media objects created in the future. The following features can be applied:
- font format: font, size, colour, style, alignment.
- background colour within the media object. These properties can also depend on those predetermined in the container that includes the media object.
- Hyperlinks colour both as hypertexts and as visited hypertexts. Further information about how to create these hypertexts can be obtained by clicking here and then hypertext.

5. Autoactions Thumb:
When opening the frame, an action such as a popout frame, an audio or a video file will be executed. For this to take place, it will be necessary to fill in the right-hand side field with the adequate Value. (See Actions)
