4. Media Object:

Media Object: it is featured by a grey square when created. It is the tool used to insert text, images, audio, video and actions such as hyperlinks and multimedia playing. It is then, an essential and basic item in the development of a frame.

1. Development: in order to add a media object, from the Frame Management, right click Insert Media in the work area

2. Edit: double click the media object or select it with the right button of your mouse and click Edit from the menu. The Media Object Edit window will open. The user can name the media in the right top field. Below, there are three thumbs which allow the user to edit the object:

2.1 Content thumb: from it the user can set up content, action and other properties:

Content:
1. Text: it allows inserting a text paragraph with line breaks which will be placed from the upper box boundary.
2. Label: the inserted text will show vertically centred and in an only line.
3. Files: in order to associate a created text, image, audio or video file to a media object, this file must have been previously saved in the corresponding folder of the teaching unit, that is, audio, image, text or video folders. If it hasn't been saved like this, the file will not appear on the corresponding list on the right hand side of the screen.

In order to save these files into the right folders, the user can manually do it from the system explorer or from the Frame Management in Malted editor: Media > Import media allows the user to browse for the adequate files in the hard disk or in a net and import them into their corresponding folders within our project.

Media Properties:
- Transparent: it turns the media background into transparent and does away with the fine border that surrounds the media object by default.
- Checkbox: it adds a check box to the left of the media that can be active or not by clicking it.
- Editable text: it turns the content of the Text window into an area which the user can edit.
- Embedded: it is only useful when there is an audio or video file associated to the media object. By clicking this option, the user of the teaching unit will be able to see a means of controlling the audio or video:


Media Action and Value:
- There is a great deal of actions that the user can apply to a media object so that when clicking on that media, a definite action takes place, for instance, to create hyperlinks within the project, open external applications, play video or audio files, … If you would like to see the full list of examples click here. Those actions which require a particular Value show instructions to fill in the corresponding field below the Value window.
 2.2 Labels Thumb: it allows the user to personalize the font outlook, its format and the background colour within the media object. These properties may depend on those set up in the container in which the media object is.

 

 
2.3 Events Thumb: it allows the user to associate a Malted Action to a mouse event, such as running the mouse over the media object (onMouseOver), right clicking the media object (onRightClick). To the right of each event, there is a box in which one must key in the corresponding value for those actions that require it. If in doubt about the syntax for a value, it can be checked by selecting the same Action from the Content thumb.
