Malted Editor (DVE) Menu - Frame Management

File Menu:

 New frame: it opens a dialogue box called New frame. The user can select a template from the menu and can also preview it before adding it to the project. _Blank template, which has no interaction associated to it, is selected by default. The name of the default frame can be kept or changed. The user can have more than one frame open at the same time.

Open…: it opens an existing frame within the project. It opens a dialogue box with a list of the frames included in the project, out of which the user can select one. You can have more than a frame open at the same time.

Close: it closes the active frame. If it hasn't been saved, the system will ask the user for confirmation to save it.

Save: it saves a frame in the hard disk with the name which the user had provided it with when creating it.

Save as …: it allows the user to save the frame with a new name, keeping a copy of the originally named frame.

Import frame: it opens a dialogue box to import frames from other projects into the one we are working with.



The Remote Import is being implemented at the moment, so the user will be able to carry out a Local Import so far; in order to do this, click Search local disks, browse for the project that contains the frame that one wants to import and select that frame from the xml folder of the project; then open it. Repeat this process as long as necessary and finally Add list to project so as to complete the import.

New project: it opens the New project dialogue box, in which the user must key in the name of the new project. This option will not be activated if there is already an open project in Malted editor. In order to create a new project, it is necessary to close the project in which one is working beforehand. The dialogue box also allows the user to choose where in the hard disk the project will be saved.

Open project: it opens a dialogue box with a list of the existing projects in the hard disk, out of which the user can select one. In order to open a project, select it and then click Open.

Close project: it closes the open project and asks for confirmation to save the courses or frames of the project which had not been saved yet.

Exit: it closes the tool. It asks for confirmation to save the courses or frames which had not been saved. It saves the application options for later use.

Edit Menu

 Undo: it undoes the latest change made to the frame in which the user is working.

Cut/Copy/Paste/Delete: basic actions applicable to all the objects within a frame (Media Object, Panels, Cards, Containers).

 Edit Script : Edit Script: It opens the frame xml code in the text editor so that it can be modified. For this option to be active, it is necessary to save all the changes made to the frame first. Once the changes to the code have been made, save the xml file and, after returning to the Malted Editor window, select Edit>Reload frame so that the changes are applied .

Reload frame: it refreshes the frame once changes to the xml code have been changed, so that those changes are shown in the edit screen.

Options: it opens the Malted Options window, which allows the user to change the language of the tool, to set up if the Malted Editor must be opened from the Frame or course Management, to add external applications to the Tools menu, and to set up the access to the Asset Base.



 Frame Menu

 The actions within this menu will only be available if there is an active frame.

Properties: it opens the Frame properties window, where the interaction and other general features are applied to the frame. For further information click here.

Description: it opens the frame properties panel, where the user can find the name of the frame and its description, both modifiable.

Run frame: it opens a preview of the frame with all its functional actions, except for those related to other frames of the course.

Media Menu

Demo Multimedia 7:"Importing Media"
Import Media: it allows the user to browse for media files outside our project so as to add them. It opens a box from which the user can select a media file from the hard disk or load a media file from an external asset base.



Click Search Local Disks and we will find the files to import .



When doubleclicking a file, it is added to the import list and the user can repeat the process with as many files as needed.



After completing the list of files to import, click Add list to project. The files of the list will be copied to the corresponding folders (image, audio, etc) of the project .

Edit: it launches an external editor to edit the selected media file. It chooses the application according to the extension of the media file, exactly as it is defined in options.

Preview: it previews the selected media file.

Rename: it opens a dialogue box with a full text field corresponding to the name of the current media file. A new name can be entered in the text field and then the user can click Accept to confirm the action.

Tools Menu

It provides a list of all the available editors (external applications), which have been previously set up from Edit>Options. Each part of this menu opens its corresponding application .

Windows Menu

Course: it opens the Course Management.

Frame: it opens the Frame Management.

At the bottom of the Windows Menu, the user can see a list of all the open frames. The active frame appears buttoned. From this list the user can swap active frames so as to edit another open frame.

Help Menu

 Malted Help: it opens a window with Malted Editor tutorial.

About: it opens a window with information about Malted project.
