3.4. El Objeto Media :


Objeto Media: Se representa con un cuadrado gris al ser creado. Es la herramienta que emplearemos para introducir texto, imágenes, audio, vídeo y acciones como vínculos y reproducción de multimedia. Es, por tanto, el elemento fundamental y básico en la creación de un frame.  


1. Creación: Para Añadir un Objeto Media, estando en el Modo de Edición de Frames: Pulsar con el botón derecho del ratón sobre la zona de trabajo > Insertar media

2. Edición: Hacemos doble clic sobre el Objeto Media o lo seleccionamos con el botón derecho y pulsamos sobre la opción Edición del menú. Se abrirá la ventana de Edición del Objeto Media. En la casilla superior podemos asignar un Nombre. Debajo hay 3 pestañas que permiten editar el objeto:

2.1 Pestaña Contenido: Desde aquí podemos asignar el Contenido, la Acción y otras propiedades.


Contenido:
1. Texto: Permite introducir un párrafo de texto con saltos de línea que partirá del borde superior de la caja.
2. Etiqueta: El texto introducido aquí aparecerá centrado verticalmente y en una sola línea .
3. Ficheros: Para asociar al Objeto Media un fichero ya creado (de texto, imagen, audio, vídeo) éste debe estar previamente en la carpeta adecuada de la Unidad en cuestión (audio, image , text, video). De otra manera, no aparecerá el archivo en el desplegable a la derecha de cada tipo de fichero:
-Fichero texto: asocia un fichero de texto
-Fichero imagen: asocia un fichero de imagen
-Fichero img. sust.: asocia una imagen de sustitución al pasar el ratón por encima del media
-Fichero Audio: asocia un fichero de sonido
-Fichero Video: asocia un fichero de vídeo.
Para introducir estos archivos en sus respectivas carpetas el usuario puede hacerlo de forma manual desde el explorador del sistema o en el Editor Malted, desde el Modo de Edición de Frames: el menú Media > Importar media permite localizar los archivos deseados en el disco duro o en el Asset Base e importarlos a sus respectivas carpetas dentro de nuestro Proyecto.

Propiedades del Media:
- Transparente: Convierte el fondo del media en transparente además de eliminar el marco fino que rodea al Objeto Media por defecto.
- Casilla de verificación: Añade una casilla de verificación en el lado izquierdo del Media, que se activa o desactiva con el botón izquierdo del ratón.
- Zona editable: Convierte el contenido de la ventana Texto en zona editable para el alumno.
- Incrustar: Sólo tiene sentido cuando hemos asociado un fichero de audio o vídeo al Objeto Media. Con esta opción marcada se muestra al alumno una barra de control del audio / vídeo:

- Barra de desplazamiento: permite seleccionar, en caso de desbordamiento, si aparecerá o no la barra de desplazamiento vertical en el media.
- Borde: permite seleccionar si el media tendrá un marco o un borde. Si el media es transparente, no aparecerá el borde aunque esté seleccionado.
- Grosor: se determina aquí el grosor del borde.
- Color: permite seleccionar el color del borde.

Acción y Valor del Media:
 - Existen multitud de Acciones que podemos asignar a un Objeto Media para que al pulsar con el ratón encima del mismo se ejecute la acción seleccionada. Crear vínculos dentro del Proyecto, abrir aplicaciones externas, ejecutar audios o vídeos, ... Para la lista completa con ejemplos pulse aquí. En aquellas acciones que requieran un Valor determinado, bajo la ventana Valor aparecerán instrucciones para rellenar esta casilla.
 2.2 Pestaña Etiquetas: Permite controlar el aspecto de la Fuente, su formato y el color de fondo dentro del Objeto Media. Estas propiedades pueden estar supeditadas a las definidas en el Contenedor que alberga al Objeto Media.

 

 
2.3 Pestaña Eventos: Permite asignar una Acción Malted a un Evento del ratón del tipo que el puntero pase por encima del Objeto Media (onMouseOver) o que se presione sobre el Objeto Media con el botón derecho (onRightClick). A la derecha de cada Evento hay una casilla donde se deberá introducir un Valor para aquellas Acciones que lo precisen. Si no estamos seguros, podemos consultar la sintaxis para este Valor seleccionando la misma Acción en la pestaña Contenido.

 


2.4. Pestaña Dimensiones: permite colocar y dimensionar el media usando campos de texto. Estos campos recogen la especificación de la posición y el tamaño del media expresados en píxeles.


3. Superposición de Objetos Media:  Tanto la Barra de herramientas estándar del DVE como el menú contextual para la edición del objetos recogen la función de enviar dicho objeto "hacia el frente" o "hacia el fondo", a fin de acomodar contenedores, paneles y objetos media en un frame cambiando su prioridad o su orden de presentación. En la Barra de herramientas estas opciones se encuentran al lado de los pulsadores "Back" y "Next" que se usan para navegar entre tarjetas.
