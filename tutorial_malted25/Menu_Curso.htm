Menú del Editor Malted (DVE) - Modo Editor de Cursos


Menú "Archivo":

 Nuevo curso: Crea un curso nuevo. Si hay algún curso activo en la ventana curso, lo cierra preguntando previamente si se desea guardarlo.

Abrir...: Abre un cuadro de diálogo con una lista de los cursos que el Proyecto contiene, de entre los cuales se puede escoger uno. A diferencia de los frames, no puede haber más de un curso activado a la vez.

Cerrar: Cierra el curso activo. Si no ha sido guardado, el sistema pregunta si se quiere guardar con la extensión correspondiente (.cxml).

Guardar: Guarda el curso con la extensión correspondiente (.cxml) en el Proyecto.

Guardar como...: Guarda el curso con la extensión correspondiente (.cxml) en el disco duro y con un nombre de archivo diferente.

Importar cursos: Abre un cuadro de diálogo para importar cursos de otros proyectos. Los cursos tienen extensión .cxml y se albergan dentro de la carpeta xml del proyecto.

Nuevo proyecto: Abre el cuadro de diálogo de "Nuevo proyecto" en el que el usuario debe introducir el Nombre del nuevo proyecto. Ésta opción está desactivada si ya hay algún Proyecto abierto en el Editor Malted. Para poder crear un Proyecto nuevo, es necesario cerrar de antemano el Proyecto en el que se esté trabajando. El cuadro de diálogo también permite elegir la ubicación en el disco duro donde se desea guardar.

Abrir proyecto: Abre un cuadro de diálogo en el que hay una lista con los proyectos existentes en el disco duro, de entre los cuales el usuario puede escoger uno. Para abrir un proyecto, seleccionarlo y luego pulsar sobre el botón Abrir.

Cerrar proyecto: Cierra el proyecto que está abierto. Pregunta si se desean guardar los cursos o los frames del proyecto que no han sido guardados.

Renombrar: Abre un cuadro de diálogo en el que se puede escribir un nombre nuevo para el curso seleccionado. Un clic sobre la opción "Aceptar" abre un cuadro de diálogo pidiendo la confirmación de la acción a ejecutar. En caso de no rellenar el campo de nombre de proyecto, aparecería en la pantalla un mensaje de error.

Salir: Cierra la aplicación. Pregunta si se desea guardar los cursos o los frames que no han sido guardados. Guarda las opciones de la aplicación para usos posteriores.

Menú "Edición"

Deshacer: Deshace el último cambio realizado en el Curso en el que se esté trabajando.

Borrar: Borra el frame seleccionado en la ventana de la izquierda

Editar Script : Abre el código cxml del Curso en el editor de texto para poder modificarlo. Para que esta opción se active, es necesario Guardar primero los últimos cambios realizados en el curso. Una vez hechos los cambios en el código, guardar el fichero en el editor de texto y al volver a la ventana del Editor Malted se debe seleccionar el menú Edición > Actualizar frame para que se reflejen los cambios en el Curso.

Actualizar Frame: Actualiza el Curso cuando hemos realizado cambios en el código.

Preferencias: Abre la ventana "Malted - Opciones", que permite cambiar el idioma de la aplicación, establecer si deseamos que al iniciar el Editor Malted se abra en Modo Editor de Frames o de Curso, añadir aplicaciones externas al menú de Herramientas, y configurar el acceso a una Base de Datos.


Menú "Curso"

Propiedades: Abre una ventana con las propiedades del curso, desde donde se puede modificar el nombre, la descripción y la estructura (lineal o compleja) del mismo.

Agregar un frame: Abre un cuadro de diálogo con la lista de plantillas. El frame seleccionado se agregará al curso activo.

Ejecutar el curso: Abre el Navegador-Editor Malted (RTE), permitiendo navegar por el curso y todos sus frames.

Create Course html: Genera un fichero html del curso necesario para poder verse desde el applet Malted de la web.

 Menú "Frame"

Importar frame: Abre un cuadro de diálogo para importar frames desde otros Proyectos a aquél en el que estamos trabajando. Cabe realizar una importación local o una importación remota.

-Importación local:


 

Hay que pulsar el botón "Buscar en discos locales", localizar el Proyecto con el frame deseado y, dentro de la carpeta xml del Proyecto, seleccionar el frame y abrirlo. Se puede repetir la operación para importar más frames a la lista y, cuando se haya terminado, se debe pulsar el botón "Agregar la lista al proyecto" para que se realice la importación.

-Importación remota

Hay que pulsar el botón "Importación remota", localizar el frame deseado y abrirlo. Se puede repetir la operación para importar más frames a la lista y, cuando se haya concluido, se debe pulsar el botón "Agregar la lista al proyecto" para que se lleve a cabo la importación.

Descripción: Abre el panel de propiedades de los frames en el que se encuentra el nombre del frame y una descripción del mismo que se puede modificar.

Editar frame: Abre el Frame seleccionado en la ventana izquierda, pasando automáticamente al Modo Editor de Frames.

Editar link: Abre una ventana de edición de enlaces entre los frames contenidos en un proyecto complejo.

Ejecutar frame: Abre el frame en modo de Previsualización con todas las Acciones funcionales, salvo aquellas que hagan referencia a otros frames del Curso.

 Menú "Modo"

Curso: Activa el Modo Editor de Cursos.

Frame: Activa el Modo Editor de Frames.

En la segunda mitad del menú Modo aparece una lista con los frames abiertos. El frame activo aparece con un punto. Desde aquí se puede cambiar de frame activo para editar otro frame que esté abierto.

Menú "Ayuda"

 Ayuda Malted: Abre una ventana con el tutorial del Editor Malted (DVE).

Acerca de...: Abre una ventana con información sobre el proyecto Malted.
