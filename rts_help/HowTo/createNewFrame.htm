How to create a new frame

  1. You must be in the Frame Management: click on the Windows menu, then the Frame item.
  2. Click on the "File" menu, then on the "New frame" item. A dialog window will appear.
  3. Select one among the templates you see. In order to help you decide, a short description appears for each template selected.
  4. Enter the title of the new frame
  5. Click OK
  6. The Cancel button closes the dialog without any action being performed
