HOW TO
 
  • Open a courseware 
  • Connect to de Asset Base
  • Exit the Application

 

Open a courseware

Normally, on opening the Run Time System a courseware selection frame will be displayed, as shown in the screen captures below.
 

If in the Index frame, click on one of the repository section buttons, then select one of the units (courseware), and finally click on the View Unit button for the opening frame of the course to be displayed. When you exit a course, the Index frame will still be there, ready for a new selection.

If in the multi-point selection frame, first select one of the modes: Local hard disk, LAN or Online. With the first two options, a user dialog interface will open as a pop-up window. By selecting the projects source in the upper section, the available projects will appear in the lower section in alphabetical order. You can then browse within the existing assets and finally select a courseware to be displayed. The Online option will connect you to the Malted remote official resources base.

At any time, you may click on the Open icon  on the navigation bar and the asset selection pop-up window will be shown again. Hence, you do not need to exit the application to view a new courseware.

 

Connect to the Asset Base

In the case the assets are to be downloaded through the network, an animated symbol shows that the downloading is in process. Once the connection with the remote AssetBase has been established, the assets are downloaded in the cache of the computer with all their associated assets (e.g. a courseware script with all its associated images and audio files). When this process is over, you will be able to run the course.

 

Exit the Application

Click on the exit icon  on the navigation bar.
