Memory

Permite hacer un ejercicio de buscar parejas tratando de recordar donde se encontraba cada elemento. Los textos o imágenes aparecen como cartas a las que se ha dado la vuelta, y solo podemos verlas de dos en dos. Cuando levantamos dos cartas emparejadas, desaparecen de la pantalla. Tenemos que ir levantando cartas y recordándolas hasta poder descubrir todas las parejas y dejar la pantalla limpia.

 

ELEMENTOS NECESARIOS

1. Panel con Nombre "Main".

2. Dentro del Panel "Main", un Contenedor con Nombre "Options". Dentro de él se incluirán los elementos que hay que emparejar (las "cartas"). Estos media pueden llevar asociado una imagen o un texto (en el campo etiqueta). Para indicar al programa cuales son las cartas que forman pareja, estas deben tener asignado el mismo número en el campo valor.

3. Media con Acción "retry" y Valor "Panel(Main)" que permite volver el ejercicio al estado original y empezar de nuevo (botón Repetir).

NOTA: Todos los demás elementos que componen esta plantilla (los botones de ayuda e instrucciones, el icono de la mano para pasar al siguiente ejercicio, etc.) no son necesarios, están ahí para servir de ejemplo. Elimina los que no vayas a usar para tu ejercicio (como por ejemplo el botón que abre esta explicación).

 

PARÁMETROS CONFIGURABLES

Tamaño de pantalla 640x480 800x600 1024x768

Pantalla completa (marcado/desmarcado) Cuando está activado ajusta los elementos a la resolución de la pantalla. Si está desactivado mantiene siempre la misma apariencia.

Sonido de respuesta: Acierto / Error (nombre del fichero) Se puede definir un fichero de sonido (se elige desde la lista desplegable) que sonará cuando conseguimos o no una pareja.

Otros parámetros:

backCardImage (image/nombre de la imagen)

Indica al programa la imagen que queremos que aparezca como parte de atrás de las cartas.
